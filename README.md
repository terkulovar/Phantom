
A. How to run
  1. Create a build directory
  mkdir build

  2. Go there
  cd build

  3. Run cmake  pointing to install directory of Geant4 and the source of project \
  cmake -DGeant4_DIR=/your_path_to/Geant4/geant4-v11.0.3-install/lib/Geant4-11.0.3 /your_path_to_the source_the_project
  or
  cmake ..

  4. make
  
  6. Running in interactive mode
     ./RunGun
